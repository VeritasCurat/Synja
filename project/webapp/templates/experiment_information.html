Information about the experiment

Thank you for visiting this site!
With my bachelor work, i investigated how a pedagogical agent and RASA Technology could be used to create a virtual help to learn Java syntax. This is the result of the Pedagogical Agent Synja (Pedagogical Agent for Learning the Syn of Java.

To investigate the Effectiveness of the Pedagogical Agent, I would ask the Participants to edit one Test before and after the Use of Synja with the Scope of 10 small Task (Effort 2-5 Minutes). Each of the Participants will receive a small Prize :).
