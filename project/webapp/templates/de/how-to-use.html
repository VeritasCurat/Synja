Das Ziel von Synja

Synja ist ein pädagogischer Agent um die Syntax von Java zu lehren. Dabei sollen Schüler die noch keine Erfahrung mit Java oder Schüler die Probleme mit Java-Syntax in ihrem Lernprozess unterstützt werden.

Die Lehrmethoden von Synja

Synja implementiert ein von mir eigens entwickeltes didaktisches Modell um die Syntax der Sprache zu lernen (ein Synthese aus dem EIS-Prinzip von Bruner, ausgearbeiteten Beispielen und einer dialogbasierten Fehlererklärung). Der Kurs ist in Themenblöcke (Programmstruktur, Grundlagen, Arrays, Operatoren, Anweisungen, Kontrollstrukturen, Methoden und Klassen) unterteilt, in welchen Konzepte unterrichtet werden. Am Ende eines Themenblockes soll der Nutzer mindestens 3 Aufgaben (Multiple-Choice-Quizze und Lückentexte) korrekt lösen um das Wissen zu festigen. Vor der Benutzung kann der Nutzer einen Pretest ausfüllen, mithilfe dessen Synja ihre Lehre an die Probleme des Schülers anpasst.

Die Benutzeroberfläche von Synja.

Die Benutzeroberfläche von Synja besteht aus drei Teilen. Links: das Dialogfeld, in dem der Dialog aufgezeichnet ist, rechts: die bildliche Darstellung von Synja und unten die beiden Eingabefelder (links: das Eingabefeld für Eingaben die sich auf den Dialog beziehen und rechts das Eingabefeld für Eingaben die sich auf Aufgaben beziehen).

Die Fähigkeiten im Dialog.

Synja kann einfache Sätze verstehen. Das parsen der natürlichen Sprache wird durch RASA Technologie ermöglicht.
